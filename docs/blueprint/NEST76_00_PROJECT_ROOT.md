# Product Definition

NEST76 is a **digital proof-of-handover system**.

It records moments when one person gives an item to another.

Example events:

* driver delivers item
* laundry delivered to reception
* courier leaves package with security
* colleague passes documents

The system captures:

WHAT
WHO
WHEN
WHERE
PROOF

---

# Core Idea

Human behavior:

Person A gives item
↓
Person B receives item
↓
Trust moment occurs

NEST records this moment digitally.

---

# Mode 0 Scope

Mode 0 focuses on the simplest possible workflow.

Create Handover
Share Link
Receiver Clicks TERIMA
Receive Event Recorded

Goal:

Create a valid handover record in under **10 seconds**.
